# **1\. アプリ概要**

## **1.1 目的・背景**

本アプリケーションは、VOICEROID2 等の合成音声ソフトで制作した自作音声ファイル、および exVOICE 等の商用音声素材を一元管理するための Windows デスクトップアプリです。

ぴた声（あっとそふと）を参考に、以下の課題を解決することを目的とします。

* 膨大な音声ファイルの散在・管理困難

* キャラクター・用途別の素早い絞り込みができない

* 商用音源と自作音声の混在による管理コスト増大

* タグ付け・メモ機能の欠如

## **1.2 参考アプリケーション**

参考：ぴた声（あっとそふと）

* https://www.ah-soft.com/pitagoe/\#app

* https://www.ah-soft.com/rs/manual/voice\_lib.html

## **1.3 対応音声ソフト・素材**

| 種別 | ソフト / 素材名 | 備考 |
| ----- | ----- | ----- |
| 自作音声（合成） | VOICEROID2 | テキスト読み上げで生成した WAV |
| 自作音声（合成） | CeVIO / A.I.VOICE 等 | 各種音声合成ソフト全般 |
| 商用音声素材 | exVOICE（VOICEROID2） | 製品付属の音声素材パック |
| 商用音声素材 | その他販売音源 | 購入したボイス素材全般 |

# **2\. 推奨技術スタック**

## **2.1 フロントエンド / フレームワーク**

| レイヤー | 採用技術（推奨） | 採用理由 |
| ----- | ----- | ----- |
| デスクトップ基盤 | Electron または Tauri | Windowsローカルファイルへのフルアクセス、WAV/MP3再生が容易 |
| UIフレームワーク | React \+ TypeScript | コンポーネント設計・型安全で大規模対応しやすい |
| スタイリング | Tailwind CSS | 高速プロトタイピング、カスタマイズ性高 |
| DB / ストレージ | SQLite（better-sqlite3） | ローカル軽量DB、全文検索対応（FTS5） |
| 音声再生 | Web Audio API / howler.js | WAV・MP3・OGG 等の再生、波形表示も可能 |
| アイコン | Lucide React | 軽量SVGアイコンセット |

*※ Electron は Node.js フルアクセス・エコシステムが豊富。Tauri は Rust ベースで軽量・高速だが学習コストがやや高い。まず Electron での実装を推奨。*

## **2.2 データ保存場所**

* 管理DB（SQLite）: %APPDATA%\\VoiceManager\\db\\voice.db

* 一括管理フォルダ（コピー先）: ユーザーが任意に設定（デフォルト: Documents\\VoiceManager\\）

* アプリ設定ファイル: %APPDATA%\\VoiceManager\\config.json

## **3.2 ファイル取り込み機能**

### **3.2.1 ドラッグ＆ドロップ取り込みフロー**

| Step | 操作 | システム動作 |
| :---: | ----- | ----- |
| 1 | D\&Dでファイル/フォルダをアプリへ | ドロップエリアを強調表示し受け付け準備 |
| 2 | ドロップ確定 | 音声ファイルを検出（WAV, MP3, OGG, FLAC, M4A） |
| 3 | カテゴリ選択ダイアログ | 「商用音源」または「自作音声」を選択させる |
| 4 | キャラクター確認 | ファイル名/フォルダ名からキャラクターを自動抽出し確認ダイアログを表示 |
| 5 | コピー実行 | 管理フォルダ（商用 or 自作）配下へファイルをコピー |
| 6 | DB登録 | ファイル情報・キャラクター・カテゴリをDBに保存 |
| 7 | 取り込み完了 | 一覧に新規ファイルが表示される |

*※ 元ファイルはコピーのみで移動・削除は行わない。管理フォルダはユーザーが設定画面で変更可能。*

### **3.2.2 対応ファイル形式**

* WAV（最優先・VOICEROID2の出力形式）

* MP3

* OGG

* FLAC

* M4A / AAC

## **3.3 商用音源・自作音声の分離管理**

アプリ内でデータを2つのカテゴリに完全分離して管理します。

| 項目 | 商用音源ライブラリ | 自作音声ライブラリ |
| ----- | ----- | ----- |
| 対象 | exVOICE・購入ボイス素材等 | 自分で合成・録音した音声 |
| 管理フォルダ | VoiceManager/commercial/ | VoiceManager/original/ |
| UI表示 | 左ペイン「商用音源」タブ | 左ペイン「自作音声」タブ |
| キャラ管理 | 製品名・バージョンも管理 | 使用ソフト名も管理 |
| 検索範囲 | 商用ライブラリ内のみ or 全体 | 自作ライブラリ内のみ or 全体 |

## **3.4 キャラクター管理**

### **3.4.1 自動抽出ロジック**

ファイルまたはフォルダ名からキャラクター名を自動判定します。判定は以下の優先順位で実施します。

| 優先度 | 判定元 | 例 | 動作 |
| :---: | ----- | ----- | ----- |
| 1 | 直属フォルダ名 | 琴葉茜\\〇〇.wav | 「琴葉茜」を抽出 |
| 2 | ファイル名プレフィクス | 結月ゆかり\_〇〇.wav | 「結月ゆかり」を抽出 |
| 3 | ファイル名サフィックス | 〇〇\_琴葉葵.wav | 「琴葉葵」を抽出 |
| 4 | キャラ辞書との部分一致 | ファイル名中にキャラ名を含む | 辞書にあれば自動判定 |
| 5 | 手動設定 | 自動抽出失敗時 | ユーザーが手動で修正 |

*※ 区切り文字として \_（アンダースコア）、-（ハイフン）、半角スペース、全角スペースを認識。*

*※ 自動抽出後、確認ダイアログでユーザーが修正・確定する。*

### **3.4.2 キャラクター辞書**

* 設定画面でキャラクター名リストを登録・編集

* 初期辞書として主要VOICEROID2キャラを同梱（結月ゆかり、琴葉茜、琴葉葵、東北ずん子、その他）

* 辞書はCSV形式でインポート/エクスポート可能

## **3.5 タグ管理**

* 各音声ファイルに対して複数タグを設定可能

* タグは自由入力（新規作成）またはプリセットから選択

* 使用頻度順・アルファベット順でタグ一覧表示

* タグをクリックして即座にフィルタリング

* 複数ファイルを選択して一括タグ付け

* タグの色分け対応（任意）

## **3.6 検索・フィルター**

### **3.6.1 検索条件**

| 検索項目 | 検索方式 | 例 |
| ----- | ----- | ----- |
| ファイル名 | 部分一致・あいまい検索 | 「ありがとう」でファイル名を絞り込み |
| タグ | 完全一致・複数AND/OR | タグ「感謝」AND「笑い声」 |
| キャラクター | 完全一致・複数選択 | 結月ゆかり・琴葉茜を選択 |
| カテゴリ | 選択 | 商用 / 自作 / 全体 |
| 音声ソフト | 選択 | VOICEROID2 / CeVIO 等 |
| ファイル形式 | 選択 | WAV / MP3 / FLAC 等 |
| 登録日 | 日付範囲 | 2025/01/01 〜 2025/03/31 |
| 再生時間 | 範囲（秒） | 1秒以上 3秒以下 |
| メモ | 部分一致 | メモ本文をキーワード検索 |

### **3.6.2 フィルターパネル**

* 画面左サイドバーにフィルターパネルを常時表示

* キャラクター・タグ・カテゴリをチェックボックスで複数選択

* よく使うフィルター条件を「フィルターセット」として保存可能

* 検索結果件数をリアルタイム表示

## **3.7 音声再生**

* ファイル一覧でファイルをクリックまたはスペースキーで即時再生

* アプリ内蔵プレイヤーで再生（外部アプリ不要）

* 再生中はプログレスバー・現在時刻・総時間を表示

* 音量調整スライダー

* 再生完了後、次のファイルへの連続再生オプション

* 波形表示（WaveSurfer.js 等を使用）

# **5\. データ構造設計**

SQLite（better-sqlite3）を使用。FTS5（全文検索）拡張を活用してファイル名・メモの高速検索を実現します。

# **6\. ファイル管理フォルダ構造**

管理フォルダ（デフォルト: Documents\\VoiceManager\\）の構造は以下の通りです。

| パス | 内容 |
| ----- | ----- |
| VoiceManager\\ | 管理フォルダルート |
| VoiceManager\\commercial\\ | 商用音源ライブラリ |
| VoiceManager\\commercial\\結月ゆかり\\ | キャラ別サブフォルダ（自動生成） |
| VoiceManager\\commercial\\結月ゆかり\\〇〇.wav | コピーされた音声ファイル |
| VoiceManager\\original\\ | 自作音声ライブラリ |
| VoiceManager\\original\\琴葉茜\\ | キャラ別サブフォルダ（自動生成） |
| VoiceManager\\original\\琴葉茜\\〇〇.wav | コピーされた音声ファイル |

*※ キャラクター不明の場合は「\_未分類」フォルダに格納。*

*※ ファイル名が重複する場合は末尾に連番を付与（例: file\_001.wav, file\_002.wav）。*
